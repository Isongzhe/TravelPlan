# 專案說明
這是一個關於旅遊規劃的網站，用戶可以藉由提供Google Map已有的清單連結、旅遊來回地、來回日期。<br>利用本網站去規劃每天景點排序，此網站能幫用戶計算景點間的路線，以及透過分圖層的方式呈現每天的行程。

# 動機
因為Google My Map實在太難用了，想自己開發一個自己用。
